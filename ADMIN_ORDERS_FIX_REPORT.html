Problem: Incorrect authentication logic

The Broken Code:

function verify(req: NextRequest) {
  const header = req.headers.get('authorization') || '';
  const token = header.startsWith('Bearer ') ? header.slice(7) : '';
  const expected = process.env.ADMIN_TOKEN || '';
  if (!expected || token !== expected) return false;  // ‚ùå WRONG
  return true;
}

Problems:

  • Ignored the PBKDF2 hashing system
  • Failed authentication check
  • Returned 401 Unauthorized
  • Frontend received error, displayed empty table
Note: A correct verification function existed in lib/auth-admin.ts but was NOT imported or used!
